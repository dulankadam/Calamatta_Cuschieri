
# CC_ChatQueue Solution

## Overview
**CC_ChatQueue** is a modular .NET solution designed to manage chat queue operations efficiently and reliably. The project follows a clean architecture pattern, ensuring scalability, maintainability, and ease of testing.

The solution includes three main projects:
- **CC_ChatQueue** – Core application logic and implementation.
- **CC_ChatQueue.Tests** – Unit test suite validating core functionalities.
- **CC_ChatQueue.Test** – Additional test project for extended or integration-level validations.

This repository is built with **.NET SDK 7.0+** and is compatible with the latest **.NET 8.0 SDK** for improved performance and long-term support.

---

## Project Structure

- **CC_ChatQueue**: Contains the main application code, including services, models, and business logic for managing chat queues.
